# Balaji Gurumurthy – Personal Profile Website

## Project Overview
This is a multi-page static website built as an assignment to demonstrate semantic HTML, organized CSS, accessibility, and responsive design. The content is based on my professional profile.

## Design Rationale

### HTML Structure
The website uses semantic HTML5 elements such as <header>, <nav>, <main>, <section>, <article>, and <footer>. This ensures logical content organization and accessibility. Navigation is inside <nav>, and each page’s main content is wrapped in <main>. Content blocks use <section> and <article> for clarity.

### CSS Layout & Organization
All styles are in css/style.css. Flexbox is used for layout (profile, skills, forms) to ensure flexible and responsive arrangements. CSS is grouped by component and follows a consistent naming convention for maintainability.

### Accessibility
Accessibility is addressed by providing meaningful alt text for all images, using a logical heading hierarchy, and associating <label> elements with form inputs. Navigation is keyboard-accessible, and color contrast meets accessibility standards.

### Responsive Design
The site uses a mobile-first approach, with styles optimized for small screens by default. Media queries adjust the layout for larger screens (desktop), such as switching the profile section to a horizontal Flexbox layout. This ensures the site is usable and visually appealing on both mobile and desktop devices.

## How to Run Locally
1. Download or clone this repository.
2. Open index.html in your browser.

## Deploying to GitHub Pages
1. Push your code to a public GitHub repository.
2. On GitHub, go to your repository settings.
3. Find the "Pages" section.
4. Set the source branch to main (or master) and the root folder (/).
5. Save. Your site will be live at https://<your-username>.github.io/<repository-name>/
